### 📢 Rules

- `스터디 진행 당일 13시`까지 문제별 폴더에 `이름.py` 파일 생성 및 `이슈 번호`를 포함한 `문제 풀이 코드, 설명, 주석이 담긴 PR`을 남깁니다.
- `스터디 진행 당일 16시`까지 코드 리뷰를 남깁니다.
- 매주 `월, 수, 금 16시`에 스터디를 진행합니다.

### 🌈 Convention Rule

**Issue** | `[플랫폼] 문제이름`  
ex) [Programmers - 고득점 Kit] 포켓몬

**PR** | `[스터디 날짜] 이름 #이슈번호`  
ex) [2023-01-25] dain

**Commit** | `커밋내용 문제이름 파일명`  
ex) Create 포켓몬 dain.py  
ex) Update README.md
